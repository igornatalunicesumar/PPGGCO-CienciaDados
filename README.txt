Olá pessoal, este é o repositório de recursos para a disciplina de Ciência de Dados do PPGGCO - UNICESUMAR - 2023.

Aqui serão disponibilizadas as bases de dados e os códigos de programação que serão utilizados durante a disciplina.


Temas que serão abordados:
- Pré-processamento de dados: Análise Exploratória, Seleção de Atributos e Redução de Dimensionalidade.
  - Identificação de valores ausentes;
  - Detecção e tratamento de outliers;
  - Visualizações Gráficas;
  - PCA - Análise de Componentes Principais.
- Modelos de Aprendizado de Máquina em Python.
- Análise e validação de resultados.

Alguns sites de Datasets:
https://archive.ics.uci.edu/datasets
https://www.kaggle.com/
